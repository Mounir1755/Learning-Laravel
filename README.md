# 💼 JobBoard App

A Laravel application that lets you **create**, **view**, **edit** and **delete**(**CRUD**) job listings.  
Perfect for companies to manage their job openings and for users to browse available positions.

## 🚀 Features

✅ Add new job listings (with title and salary)  
✏️ Edit or update existing jobs  
🗑️ Delete jobs you no longer need  
👀 View detailed job information  
💾 Store everything securely using Laravel Eloquent ORM  


## 🧩 Tech Stack

- **Laravel 11** – PHP framework  
- **Blade** – Templating engine  
- **MySQL** – Database  
- **Tailwind** – Front-end styling 
- **Git** – Version control
